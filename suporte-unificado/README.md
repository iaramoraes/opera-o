# Suporte Unificado — Seazone Operação

## Visão
Unificar 18 canais de suporte em 1 canal Slack (#suporte-operacao) com 4 áreas, Kanban Lovable, N8N e IA integrada.

## Estrutura

```
suporte-unificado/
├── docs/          # Documentação do projeto (decisões, specs, guias)
├── n8n/           # Workflows N8N (JSONs exportados)
├── lovable/       # Config e specs da aplicação Lovable
├── slack/         # Templates de mensagens, formulários, canvas
├── ia/            # Prompts, base de conhecimento, configs IA
└── README.md
```
