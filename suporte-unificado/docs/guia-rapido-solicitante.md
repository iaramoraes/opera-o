# Guia Rápido — Solicitante
## Como abrir e acompanhar suportes no Suporte Unificado

### Como abrir um suporte

1. Acesse o canal **#suporte-operação** no Slack
2. Clique em **"Abrir Novo Suporte"**
3. Preencha:
   - **Código do imóvel** (ex: TXP0605). Se não tiver, marque "Sem código"
   - **Área** (Implantação, Franquias, Hóspede ou Proprietários). Se não souber, marque "Não sei qual área"
   - **Processo** (aparece conforme a área selecionada)
   - **Campos específicos** do processo (varia por processo)
   - **Urgência** (Crise: 4h | Alta: 12h | Média: 24h | Baixa: 24h)
   - **Descrição** do problema
4. Clique **"Criar Suporte"**

### O que acontece depois

- Você recebe uma **DM automática** com o número do suporte e link
- Uma **thread** é criada no canal #suporte-operação
- O **responsável do fluxo** é notificado
- A cada mudança de status, você recebe uma atualização

### Como acompanhar

**No Slack:** verifique suas DMs — o bot envia atualizações automáticas.

**No app (Lovable):** acesse https://seazone-support-hub.lovable.app
- Vá em **"Meus Suportes"** para ver todos os seus suportes
- Filtros disponíveis: status, área, data

### Como cobrar andamento

- Comente na **thread do Slack** do suporte
- O comentário aparece automaticamente no card da Lovable
- O responsável é notificado

### Importante

- **Duplicidade bloqueada:** se já existe suporte aberto para o mesmo imóvel + processo, o sistema não permite abrir outro. Você recebe o link do existente.
- **SLA não reinicia** se o card for transferido entre áreas.
- **Não precisa procurar no canal.** As notificações vêm até você.
